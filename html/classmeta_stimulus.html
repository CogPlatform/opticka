Detailed Description

metaStimulus is a wrapper for opticka stimuli METASTIMULUS a collection of stimuli, wrapped in one structure. It allows you to reat a group of heterogenous stimuli as if it is a single stimulus, so for example animate(metaStimulus) will run the animate method for all stimuli in the group without having to call it for each stimulus. You can also pick individual stimuli by using cell indexing of this object. So for example metaStimulus{2} actually calls metaStimulus.stimuli{2}.
